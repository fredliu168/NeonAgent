NeonAgent

          

            NeonAgent 是一款集成大语言模型能力的 Chrome 浏览器智能体插件，旨在通过 AI 能力和网页交互技术，为用户提供强大的自动化辅助工具。
          

          

            开源地址：
            
              https://github.com/fredliu168/NeonAgent
            
          

          

            
            扫码查看项目或联系方式
          

        


        

          
🧠 核心能力

          

            
浏览器智能体 — 基于 Agent Loop 架构，智能体可自主决策并调用 24 个内置工具与网页交互，包括页面读取、元素点击、表单填写、脚本执行、导航等。

            
智能对话 — 支持 OpenAI 兼容的多种大模型接入，Markdown 渲染、实时流式输出，多会话管理与历史持久化。

            
自动答题 — 一键解题模式，AI 自动识别页面题目并填充答案。

          

        


        

          
💡 记忆系统

          

            
智能体可主动保存用户偏好、操作经验等信息，跨会话持久可用。

            
当记忆超过 50 条时自动调用 LLM 智能压缩合并，减少 token 占用。

            
支持 Markdown 格式导入/导出，方便备份和迁移。

          

        


        

          
📦 技能系统

          

            
将多步骤操作保存为可复用技能，支持一键执行、版本自动递增、使用统计。

            
支持 Markdown 格式编辑、导入/导出，方便团队共享和迁移。

          

        


        

          
⏰ 定时任务

          

            
支持创建单次、间隔、每日、每周四种调度类型的定时任务。

            
任务触发时自动启动智能体执行预设指令，适用于签到、定期检查等场景。

          

        


        

          
🛡️ 环境破解

          

            
解除右键限制、页面文字选择限制、切屏检测绕过、插件检测对抗。
